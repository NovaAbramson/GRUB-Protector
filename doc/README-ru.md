## Использование
Запустите следующие команды от пользователя (не root) в группе «sudoers» или «wheel» и следуйте инструкциям.

    git clone https://github.com/NovaAbramson/GRUB-Protector
    cd GRUB-Protector
    sudo ./Start.sh
